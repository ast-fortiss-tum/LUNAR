# SYSTEM INTENTS & EXAMPLES

SYSTEM_INTENTS_INFO = """        Intent definitions:
                                 - inform: System provides information related to the user's query
                                 - inform_and_followup: System provides information related to the user's query and asks a follow-up question
                                 - inform_repeat: System provides the information to the repeated user's query
                                 - inform_repeat_and_followup: System provides the information to the repeated user's query and asks a follow-up question
                                 - confirmation: System confirms or agrees and starts the required action; system DOES NOT provide follow-up questions
                                 - confirmation_and_followup: System confirms or agrees and starts the required action; system provides a follow-up question
                                 - clarify: System asks for clarification
                                 - choice: System offers specific options to choose from, it proposes at least 2 options to choose from
                                 - choice_and_followup: System offers specific options to choose from, it proposes at least 2 options to choose from; system provides a follow-up question
                                 - reject: System cannot fulfill request due to error, lack of information, legal or political constraints
                                 - reject_and_followup: System cannot fulfill request due to error, lack of information, legal or political constraints; system provides a follow-up question
                                 - stop: System ends conversation, possibly by providing a few options the user can do manually
                                 - misc: System's response doesn't fit into any other category

                                 Examples:

                                 System: "I'm sorry, but I can't assist with that."
